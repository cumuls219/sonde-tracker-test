# Sonde Tracker RAW v6 Windows Preview

Windows 로컬 사전 검토용 버전입니다.

## 주요 기능
- UPP RAW TXT 업로드
- GPS 기반 3D Sonde Tracker
- 실제 고도 / Skew-T형 Log-P 축 선택
- 3D 바닥면 실제 지도 표시: OpenStreetMap 정적 타일 1장 합성 후 Mesh3d로 표시
- 하늘색 계열 3D 배경
- 오른쪽 바람 벡터 패널
- 오른쪽 구름 영역 패널
- 열역학, 바람, 호도그래프, CSV/HTML 다운로드

## 실행
```powershell
python -m pip install -r requirements.txt
python -m streamlit run app.py
```

## 참고
- 바닥면 지도는 외부 인터넷 연결이 필요합니다.
- 지도 출처: © OpenStreetMap contributors
- 지도 표시가 실패해도 앱은 지도 없이 계속 실행됩니다.


업데이트: 구름 영역 패널을 제거하고, 구름 가능층은 3D Tracker에서 직접 강조합니다. 사이드바에서 구름 강조 투명도를 조절할 수 있습니다.


## v6.1 수정사항
- 사이드바에서 구름 조절, 강조층/바람, 바닥면 지도·격자 항목을 분리했습니다.
- 실제 지도 호출 실패 또는 지도 OFF 상태에서도 바닥면 기본 격자가 표시되도록 수정했습니다.
- 3D 바닥면 지도 trace가 실제로 그래프에 추가되도록 수정했습니다.


## v6.2 수정
- 지도 ON/OFF와 무관하게 바닥면 기준판과 진한 격자가 표시되도록 수정
- Log-P 모드에서 격자가 최저 z축에 묻히지 않도록 바닥면 높이와 z축 범위를 보정
- 지도 호출 실패 시에도 기준 격자만으로 수평 위치를 확인 가능


업데이트 v6.3: 바닥면 격자를 예시 모식도처럼 얇게 조정하고, 3D 옆면/연직면에 하늘색 배경과 연직 격자가 보이도록 보정했습니다. 오른쪽 바람 패널은 벡터에서 기상청식 바람깃(5/10/50 kt)으로 변경했습니다.


## v6.4 수정

- 지도 표시 옵션과 연동되는 하늘색 옆면/후면 패널 추가

업데이트 v6.5.1: Plotly 최신 버전에서 지원하지 않는 axis titlefont 속성을 title.font 구조로 수정하여 바람깃 패널이 정상 표시되도록 보정했습니다.

업데이트 v6.7: 3D Tracker 영역을 넓히고, 오른쪽 바람깃을 기상청식 연직 일직선 기준의 짧은 2D 바람깃으로 재조정했습니다.

업데이트: Skew-T형 Log-P 모드에서 3D Tracker 표출 화면 세로 높이를 확장하고, 장면 영역을 위아래로 넓혀 잘림을 줄였습니다.

업데이트(v6.8.1): 운영 기본값을 가볍게 조정했습니다. 지도는 기본 지역명 선명, Log-P 모드에서는 고해상도/표면 품질 자동 경량화, 상승 애니메이션은 탭에서 체크했을 때만 생성됩니다.


업데이트: 별도 애니메이션 탭 대신 3D Tracker 오른쪽에 현재 위치 표시 On/Off와 시간 슬라이더를 배치했습니다. 현재까지의 이동 경로, 현재 존데 위치(풍선 아이콘), 바닥면 투영선을 3D 본 화면에 함께 표시합니다.


업데이트: 3D Tracker 옆 상승 이동 표시 영역에 ▶ 시작, ⏸ 정지, ⏮ 처음, ⏭ 다음 버튼을 추가했습니다.


업데이트: 자동재생 버튼은 제거하고, 처음/이전/다음/끝 및 시간 슬라이더 중심의 클릭형 상승 이동 컨트롤로 바꾸었습니다. 버튼 스타일을 정리해 가독성을 높였습니다.


업데이트 v6.8.5: 별도 상승 애니메이션 탭을 Plotly 내부 프레임 방식으로 추가했습니다. 메인 3D를 재실행하며 움직이는 방식이 아니라 경량 애니메이션 전용 Figure를 사용하며, 전체 경로·누적 trajectory·현재 존데 위치만 표시합니다.


업데이트: 3D Tracker 본체의 인라인 애니메이션을 제거하고, 상승 애니메이션을 별도 탭으로 분리했습니다. 애니메이션에서는 풍선+존데 상징을 사용합니다.


업데이트: 애니메이션 존데 풍선/RS 표시 크기를 줄이고, Plotly 애니메이션 재생 시 카메라 시점이 초기화되지 않도록 frame layout 갱신을 제거하고 uirevision을 적용했습니다.

업데이트: 애니메이션의 현재 위치 상징을 실제 존데 느낌(아이보리 풍선, 테더, RS 본체, 작은 센서/안테나)으로 보강했습니다.


업데이트: 지도/3D 공간 여유 범위 옵션을 추가하고 기본을 넓게 조정했습니다. Log-P와 실제 고도 모두 궤적 주변만 딱 자르지 않고 주변 지명을 가늠할 수 있게 표시합니다.


업데이트: 지도/공간 여유 범위에 더해 남북(위도) 방향 추가 확대 옵션을 추가했습니다. 동서보다 남북 위치감을 더 넓게 보이도록 지도 범위, 3D 축, 격자, 애니메이션 범위에 반영됩니다.
