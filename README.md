# Sonde Tracker RAW v6 Windows Preview

Windows 로컬 사전 검토용 버전입니다.

## 주요 기능
- UPP RAW TXT 업로드
- GPS 기반 3D Sonde Tracker
- 실제 고도 / Skew-T형 Log-P 축 선택
- 3D 바닥면 실제 지도 표시: OpenStreetMap 정적 타일 1장 합성 후 Mesh3d로 표시
- 하늘색 계열 3D 배경
- 오른쪽 바람 벡터 패널
- 오른쪽 구름 영역 패널
- 열역학, 바람, 호도그래프, CSV/HTML 다운로드

## 실행
```powershell
python -m pip install -r requirements.txt
python -m streamlit run app.py
```

## 참고
- 바닥면 지도는 외부 인터넷 연결이 필요합니다.
- 지도 출처: © OpenStreetMap contributors
- 지도 표시가 실패해도 앱은 지도 없이 계속 실행됩니다.


업데이트: 구름 영역 패널을 제거하고, 구름 가능층은 3D Tracker에서 직접 강조합니다. 사이드바에서 구름 강조 투명도를 조절할 수 있습니다.


## v6.1 수정사항
- 사이드바에서 구름 조절, 강조층/바람, 바닥면 지도·격자 항목을 분리했습니다.
- 실제 지도 호출 실패 또는 지도 OFF 상태에서도 바닥면 기본 격자가 표시되도록 수정했습니다.
- 3D 바닥면 지도 trace가 실제로 그래프에 추가되도록 수정했습니다.


## v6.2 수정
- 지도 ON/OFF와 무관하게 바닥면 기준판과 진한 격자가 표시되도록 수정
- Log-P 모드에서 격자가 최저 z축에 묻히지 않도록 바닥면 높이와 z축 범위를 보정
- 지도 호출 실패 시에도 기준 격자만으로 수평 위치를 확인 가능


업데이트 v6.3: 바닥면 격자를 예시 모식도처럼 얇게 조정하고, 3D 옆면/연직면에 하늘색 배경과 연직 격자가 보이도록 보정했습니다. 오른쪽 바람 패널은 벡터에서 기상청식 바람깃(5/10/50 kt)으로 변경했습니다.


## v6.4 수정
- 바닥 격자와 연직 격자 가시성 보정
- 지도 표시 옵션과 연동되는 하늘색 옆면/후면 패널 추가
- 사이드 바람 패널을 2D 바람깃 형태로 재조정


업데이트 v6.5: 오른쪽 연직 바람깃 패널을 기상청식 2D 표시처럼 크게 보이도록 조정하고, 흰 배경/고대비 선으로 가시성을 개선했습니다.


업데이트 v6.5.1: Plotly 최신 버전에서 지원하지 않는 axis titlefont 속성을 title.font 구조로 수정하여 바람깃 패널이 정상 표시되도록 보정했습니다.


v6.6 수정: 색상범례와 기호범례 겹침 보정, 구름 강조를 실제 trace opacity 방식으로 수정, 연직 바람깃을 2D 수평형 기상청식 프로파일 형태로 보정.


업데이트 v6.7: 3D Tracker 영역을 넓히고, 오른쪽 바람깃을 기상청식 연직 일직선 기준의 짧은 2D 바람깃으로 재조정했습니다.


업데이트: 연직 바람깃 패널의 세로 길이를 줄여 3D 본체와의 비율을 더 자연스럽게 조정했습니다.


업데이트: Skew-T형 Log-P 모드에서 3D Tracker 표출 화면 세로 높이를 확장하고, 장면 영역을 위아래로 넓혀 잘림을 줄였습니다.
