# v6.8.16 Terrain Relief Map

## 변경 요약

- 3D 바닥면 실제 지도 mesh 자체에 z값을 부여해 지도 위에 입체 지형감을 반영
- 사이드바에 `지도 위 입체 지형 표시` on/off 추가
- `입체 지형 높이` 슬라이더 추가: 0.00은 평면 지도, 값이 클수록 산지 기복 강조
- `지도 위 지명 직접 표시` on/off 추가
- 3D Tracker와 상승 애니메이션 모두 동일한 입체 지도/지명 옵션 반영

## 주의

- 외부 DEM 파일을 새로 요구하지 않는 경량 구현입니다.
- OSM 지도 타일 위에 내장 지형 모델을 적용해 지형감을 시각화합니다.
- 실제 DEM 기반 정밀 고도값이 필요한 경우 별도 DEM 파일 연동 버전으로 확장해야 합니다.
